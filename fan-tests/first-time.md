 - Стресс-тест:
```bash
pacman -S stress-ng
stress-ng --cpu 12 --timeout 120
```

 - Найти сенсоры, найти текущие обороты:
```bash
# Да на все вопросы
sensors-detect

# Покажет все порты для кулеров, и обороты тех, на которых че-то подключено
sensors | grep RPM
```

В выводе сенсоров будет что-то подобное:
```bash
fan1:                  1084 RPM  (min =    0 RPM)
fan2:                   844 RPM  (min =    0 RPM)
fan3:                     0 RPM  (min =    0 RPM)
fan4:                  1146 RPM  (min =    0 RPM)
fan5:                     0 RPM  (min =    0 RPM)
```

Понятно, что из 5 разьемов используется 3. В данном случае:
- 2 кулера сверху (__fan1__);
- Кулер ЦП (__fan2__);
- 2 кулера снизу (__fan4__).

После надо найти, где располагаются файлы для управления кулерами:
```bash
# Здесь располагаются директории, которые содержат сенсоры
[root@eos ~]$ ls -a /sys/class/hwmon/
.  ..  hwmon0  hwmon1  hwmon2

# В моем случае файлы для управления кулерами лежат в hwmon2
[root@eos ~]$ ls -la /sys/class/hwmon/hwmon2/ | grep pwm1_mode
-rw-r--r-- 1 root root 4096 ноя 18 16:39 pwm1_mode
```

В найденной директории будут файлы для каждого кулера:
> Лучшее описание по всему, что здесь лежит - [[Source]](https://www.kernel.org/doc/Documentation/hwmon/nct6775)<br>
> `N` - номер кулера
- `pwmN` - файл со значнием от 0 до 255 (0% и 100%), задают скорость кулера
- `pwmN_enable` - задает режим УПРАВЛЕНИЯ кулера:
    - `0` - управление отключено, максимальные обороты;
    - `1` - ручное управление, задается через файл `pwmN`;
    - `2` - режим "Тепловой круиз"; [[source]](https://www.kernel.org/doc/Documentation/hwmon/nct6775#:~:text=Thermal%20Cruise%20mode%20(2))
    - `3` - режим "Круиз скорости"; __[не использовать]__
    - `4` - Smart Fan III (nc6775f);
    - `5` - Smart Fan IV.
- `pwmN_mode` - режим РАБОТЫ кулера:
    - `0` - работа без ШИМ;
    - `1` - работа с ШИМ;
