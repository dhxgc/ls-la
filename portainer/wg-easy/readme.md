Пароль работает только в .env файле, иначе не але =)

В `compose.yml` проброс портов с указанием ip работает только с ip-адресами, dns имена указывать нельзя.

так можно:
```yaml
    ports:
      - "30301:30301/udp"
      - "127.0.0.1:30302:30302/tcp"
```

так нельзя:
```bash
    ports:
      - "30301:30301/udp"
      - "localhost:30302:30302/tcp"
```
